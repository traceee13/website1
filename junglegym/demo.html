Using Hn tags

This section is all about the differet kind of text types! Keep in mind that your titles must be properly organized. If they aren’t, readers, robots or even you will get lost in the structure! Here’s how Hn tags should be used:

< h1 > DAI LESSON < /h1 >

< h2 > BASIC OF TEXT < /h2 >

< h3 > Hn tags < /h3 >

< h3 > Common Errors < /h3 >

< h2 > BASIC OF IMAGE < /h2 >

< h3 > changing links < /h3 >

< h3 > changing properties < /h3 >

Use the different Hn tags to draw attention.
